# PyExample

Python example project

## Run

```
./app.py 
```

With any parametr output is quit different:

```
./app.py --foo
```

## Run tests

```
python3 -m unittest
```

## Install

```
sudo ./setup.py install
```

Install app into:

```
/usr/lib/python3.4/site-packages/
```

And allow to run by command: `app.py`
